TelemetryPacket — Downlink (OCS → GCS, sent to port 8080)

#[derive(Debug, Clone, Copy, serde::Serialize, serde::Deserialize)]
pub struct TelemetryPacket {
    pub seq_no:          u32,    // monotonically incrementing, per-sensor
    pub timestamp_us:    u64,    // Instant at sender, microseconds since sim start
    pub packet_type:     PacketType,
    pub sensor_id:       SensorId,
    pub priority:        u8,     // 1=highest, 3=lowest
    pub value:           f64,    // sensor reading (temperature C, watts, rad/s)
    pub is_corrupted:    bool,   // set true when fault injection corrupts this packet
    pub payload:         [u8; 64], // fixed padding / future use — initialise to 0
}
 
impl TelemetryPacket {
    pub fn new(seq_no: u32, ts_us: u64, sensor_id: SensorId, value: f64) -&gt; Self {
        let priority = match sensor_id {
            SensorId::Thermal =&gt; 1,
            SensorId::Power   =&gt; 2,
            SensorId::Imu     =&gt; 3,
        };
        Self { seq_no, timestamp_us: ts_us, packet_type: PacketType::SensorData,
               sensor_id, priority, value, is_corrupted: false, payload: [0u8; 64] }
    }
}

3.1 main.rs — Entry Point
The main function must do the following in exact order:
1. Parse CLI arguments using clap: --gcs-ip &lt;IP&gt;, --gcs-port &lt;u16&gt;, --my-port &lt;u16&gt;, --duration-s &lt;u64&gt;

9. Spawn ALL tasks (see list below). Each task gets a clone of the relevant Arcs and a CancellationToken clone
10. Set a tokio::time::sleep(Duration::from_secs(duration_s)) as the simulation timer

3.2 buffer.rs — Sensor Priority Buffer
This module implements a bounded, priority-ordered buffer using BinaryHeap. It is the central data structure that all sensor tasks write to and the downlink task reads from.

SensorReading Struct (stored in buffer)

#[derive(Debug, Clone)]
pub struct SensorReading {
    pub packet:          TelemetryPacket,
    pub buffer_insert_us: u64,  // Instant when inserted — for latency = insert - sensor_read
}
 
// BinaryHeap is a max-heap. We want lowest priority number = highest importance.
// So we invert the comparison: priority=1 (thermal) must sort HIGHEST.
impl Ord for SensorReading {
    fn cmp(&amp;self, other: &amp;Self) -&gt; Ordering {
        // LOWER priority number = HIGHER BinaryHeap priority
        other.packet.priority.cmp(&amp;self.packet.priority)
            .then_with(|| other.packet.timestamp_us.cmp(&amp;self.packet.timestamp_us))
    }
}
impl PartialOrd for SensorReading { fn partial_cmp(&amp;self, other: &amp;Self) -&gt; Option&lt;Ordering&gt; { Some(self.cmp(other)) } }
impl PartialEq  for SensorReading { fn eq(&amp;self, other: &amp;Self) -&gt; bool { self.packet.priority == other.packet.priority } }
impl Eq for SensorReading {}


SensorBuffer Struct

pub struct SensorBuffer {
    heap:     BinaryHeap&lt;SensorReading&gt;,
    capacity: usize,
    pub stats: BufferStats,
}
 
pub struct BufferStats {
    pub total_inserted:  u64,
    pub total_dropped:   u64,
    pub peak_fill:       usize,

}


SensorBuffer Methods — implement all of these

impl SensorBuffer {
    pub fn new(capacity: usize) -&gt; Self
    // Returns: None if inserted normally,
    //          Some(dropped_reading) if buffer was full and a low-priority item was evicted.
    // Logic:
    //   1. If heap.len() &lt; capacity: push directly.
    //   2. If heap.len() &gt;= capacity AND new reading priority &lt; lowest in heap:
    //      → find and remove the LOWEST priority item (highest priority NUMBER) and push new one.
    //   3. If heap.len() &gt;= capacity AND new reading priority &gt;= lowest: drop new reading.
    //   4. After any drop: log with tracing::warn!, include timestamp and sensor_id.
    //   5. If heap.len() &gt;= (capacity * BUFFER_DEGRADED_PCT as usize): set stats.degraded_mode = true.
    pub fn push(&amp;mut self, reading: SensorReading, sim_start: &amp;Instant) -&gt; Option&lt;SensorReading&gt;
 
    // Pop the highest-priority (lowest priority number) reading.
    pub fn pop(&amp;mut self) -&gt; Option&lt;SensorReading&gt;
 
    pub fn len(&amp;self) -&gt; usize
    pub fn fill_pct(&amp;self) -&gt; f64
    pub fn is_degraded(&amp;self) -&gt; bool
}


3.3 sensor.rs — Sensor Acquisition Tasks
Three independent async functions, each running in its own tokio::spawn. They simulate sensor hardware reading values and inserting into the shared buffer.

⚠ RULE [Wk2+Wk8]: Every sensor task MUST use Instant::now() — NEVER std::thread::sleep or SystemTime. Sleeping with tokio::time::sleep_until(next_deadline) is the correct approach to hit the period. Measure actual_start = Instant::now() at the TOP of each loop iteration BEFORE doing any work.

Common Pattern for All Sensor Tasks

pub async fn run_thermal_sensor(
    buffer:     Arc&lt;Mutex&lt;SensorBuffer&gt;&gt;,
    sim_start:  Arc&lt;Instant&gt;,
    state:      Arc&lt;Mutex&lt;SystemState&gt;&gt;,
    cancel:     CancellationToken,
    heartbeat:  Arc&lt;AtomicU64&gt;,  // watchdog heartbeat — update every loop
) {
    let period = Duration::from_millis(THERMAL_PERIOD_MS);
    let mut next_deadline = Instant::now() + period;
    let mut seq: u32 = 0;
    let mut consecutive_miss: u32 = 0;
    let mut hist = hdrhistogram::Histogram::&lt;u64&gt;::new(3).unwrap(); // jitter histogram
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; { tracing::info!(&quot;thermal_sensor: cancelled&quot;); break; }
            _ = tokio::time::sleep_until(next_deadline.into()) =&gt; {}
        }
 
        // --- Timing measurement ---
        let actual_start = Instant::now();
        let expected_start_us = /* compute from sim_start + seq * period */;
        let actual_start_us   = sim_start.elapsed().as_micros() as u64;
        let drift_us          = actual_start_us as i64 - expected_start_us as i64;
        let jitter_us         = drift_us.unsigned_abs();
        hist.record(jitter_us).ok();
 
        // --- Jitter alert ---
        if jitter_us &gt; THERMAL_JITTER_LIMIT_US {
            tracing::warn!(sensor=&quot;thermal&quot;, jitter_us, &quot;JITTER EXCEEDED 1ms limit&quot;);
        }
 
        // --- Simulate sensor read ---
        // Thermal: random value in range 20.0..80.0 °C
        // Power:   random value in range 0.5..5.0 W
        // IMU:     random value in range -0.1..0.1 rad/s
        let value: f64 = /* generate simulated reading */;
 
        let ts_us = sim_start.elapsed().as_micros() as u64;
        let packet = TelemetryPacket::new(seq, ts_us, SensorId::Thermal, value);
 
        // --- Safety alert: 3 consecutive misses ---
        // A &apos;miss&apos; is when jitter_us &gt; THERMAL_JITTER_LIMIT_US * 5 (severe late)
        // Increment consecutive_miss on severe miss, reset on good read.
        // If consecutive_miss &gt;= THERMAL_MISS_ALERT: log safety alert, set state to Fault.
 
        // --- Insert into buffer ---
        let insert_us = sim_start.elapsed().as_micros() as u64;
        let reading = SensorReading { packet, buffer_insert_us: insert_us };
        let latency_us = insert_us - ts_us; // sensor_read_to_buffer latency
 
        {
            let mut buf = buffer.lock().await;
            if let Some(dropped) = buf.push(reading, &amp;sim_start) {
                tracing::warn!(dropped_sensor=?dropped.packet.sensor_id,
                               dropped_seq=dropped.packet.seq_no,
                               &quot;Buffer full: dropped packet&quot;);
            }
        }
 
        tracing::info!(
            sensor=&quot;thermal&quot;, seq, value, drift_us, jitter_us, latency_us,
            &quot;sensor_read&quot;
        );
 
        // Update watchdog heartbeat
        heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
 
        seq += 1;
        next_deadline += period;
    }
 
    // Print final histogram stats
    tracing::info!(sensor=&quot;thermal&quot;, p50=hist.value_at_percentile(50.0),
                   p99=hist.value_at_percentile(99.0), max=hist.max(),
                   &quot;thermal_sensor final jitter stats&quot;);
}

3.4 scheduler.rs — Rate Monotonic Scheduler (RMS)
This task simulates the onboard task scheduler. It manages three scheduled tasks: ThermalControl (highest priority, preempts others), DataCompression, and HealthMonitor.

ℹ RMS: Rate Monotonic Scheduling (Week 10): assign priority by period. Shortest period = highest priority. ThermalCtrl (100ms) &gt; DataCompress (500ms) &gt; HealthMonitor (1000ms). Schedulability test: U = Σ(C_i/T_i) ≤ ln(2) ≈ 0.693. Assume C_i values: ThermalCtrl=5ms, DataCompress=20ms, HealthMonitor=50ms. U = 5/100 + 20/500 + 50/1000 = 0.05 + 0.04 + 0.05 = 0.14. System IS schedulable.

ScheduledTask Struct

#[derive(Debug, Clone)]
pub struct ScheduledTask {
    pub name:       &amp;&apos;static str,
    pub priority:   u8,         // 1=highest
    pub period_ms:  u64,
    pub wcet_ms:    u64,        // simulated worst-case execution time
    pub deadline_ms: u64,       // = period (implicit deadline)
    pub next_release: Instant,
    pub exec_count:  u64,
    pub miss_count:  u64,
}


run_rms_scheduler() Logic

// The scheduler wakes every 10ms (its &apos;tick&apos;).
// On each tick:
//   1. Check which tasks have reached their next_release time.
//   2. Among ready tasks, pick the one with LOWEST priority number (highest urgency).
//   3. &apos;Execute&apos; it: tokio::time::sleep(Duration::from_millis(task.wcet_ms)).await
//      — this simulates CPU consumption.
//   4. ThermalControl MUST preempt: if ThermalControl becomes ready while DataCompress
//      or HealthMonitor is &apos;executing&apos;, cancel their sleep via tokio::select! and
//      run ThermalControl first.
//   5. After execution: measure scheduling_drift_us and log TaskMetrics.
//   6. If actual_finish &gt; task.next_release + task.deadline_ms: log deadline violation.
//   7. Update next_release for the executed task.
//
// Implement preemption using tokio::select!:
// tokio::select! {
//     _ = tokio::time::sleep(wcet) =&gt; { /* normal completion */ }
//     _ = thermal_ready_signal.notified() =&gt; { /* preempted — reschedule current task */ }
// }
//
// CPU utilisation: track total_active_ticks vs total_ticks.
// Log CPU% every 10 seconds.

    state:      Arc&lt;Mutex&lt;SystemState&gt;&gt;,
    cancel:     CancellationToken,
    heartbeat:  Arc&lt;AtomicU64&gt;,
) {
    // 1. Bind socket with timeout — enforce DOWNLINK_INIT_TIMEOUT_MS (5ms)
    let socket = tokio::time::timeout(
        Duration::from_millis(DOWNLINK_INIT_TIMEOUT_MS),
        tokio::net::UdpSocket::bind(&quot;0.0.0.0:0&quot;)
    ).await;
    let socket = match socket {
        Ok(Ok(s))  =&gt; s,
        Ok(Err(e)) =&gt; { tracing::error!(&quot;Downlink socket bind failed: {}&quot;, e); return; }
        Err(_)     =&gt; { tracing::error!(&quot;MISSED COMM: downlink socket init exceeded 5ms&quot;); return; }
    };
 
    // 2. Main loop — run every DOWNLINK_WINDOW_MS (30ms)
    let mut interval = tokio::time::interval(Duration::from_millis(DOWNLINK_WINDOW_MS));
    let mut tx_seq: u32 = 0;
    let mut hist = hdrhistogram::Histogram::&lt;u64&gt;::new(3).unwrap();
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; { break; }
            _ = interval.tick() =&gt; {}
        }
 
        let window_start = Instant::now();
 
        // 3. Degraded mode: if buffer &gt; 80%, only send P1 (thermal) packets.
        let degraded = { buffer.lock().await.is_degraded() };
 
        // 4. Drain buffer — pop up to 10 packets per window
        let mut sent = 0u32;
        for _ in 0..10 {
            let reading = { buffer.lock().await.pop() };
            let reading = match reading { Some(r) =&gt; r, None =&gt; break };
 
            // Skip low priority in degraded mode
            if degraded &amp;&amp; reading.packet.priority &gt; 1 { continue; }
 
            // 5. Serialize with bincode (near-zero parse time — Wk8)
            let bytes = match bincode::serialize(&amp;reading.packet) {
                Ok(b)  =&gt; b,
                Err(e) =&gt; { tracing::error!(&quot;Serialize failed: {}&quot;, e); continue; }
            };
 
            // 6. Send with deadline enforcement
            let send_result = tokio::time::timeout(
                Duration::from_millis(DOWNLINK_WINDOW_MS),
                socket.send_to(&amp;bytes, gcs_addr)
            ).await;
 
            let queue_latency_us = sim_start.elapsed().as_micros() as u64
                - reading.buffer_insert_us;
            hist.record(queue_latency_us).ok();
 
            match send_result {
                Ok(Ok(_)) =&gt; {
                    sent += 1;
                    tracing::info!(tx_seq, sensor=?reading.packet.sensor_id,
                                   queue_latency_us, &quot;downlink_tx: sent&quot;);
                }
                _ =&gt; { tracing::warn!(&quot;downlink_tx: send timeout/error&quot;); }
            }
            tx_seq += 1;
        }
 
        let elapsed_ms = window_start.elapsed().as_millis();
        if elapsed_ms &gt; DOWNLINK_WINDOW_MS as u128 {
            tracing::warn!(elapsed_ms, &quot;downlink_tx: exceeded 30ms window&quot;);
        }
        heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
    }
    tracing::info!(p50=hist.value_at_percentile(50.0), p99=hist.value_at_percentile(99.0),
                   &quot;downlink_tx final queue latency stats&quot;);
}


3.6 uplink_rx.rs — Command Receiver from GCS

pub async fn run_uplink_rx(
    my_addr:    SocketAddr,     // 0.0.0.0:9090
    state:      Arc&lt;Mutex&lt;SystemState&gt;&gt;,
    sim_start:  Arc&lt;Instant&gt;,
    cancel:     CancellationToken,
    heartbeat:  Arc&lt;AtomicU64&gt;,
) {
    // Bind to the uplink port — GCS sends commands here
    let socket = UdpSocket::bind(my_addr).await.expect(&quot;Uplink RX bind failed&quot;);
    let mut buf = [0u8; UDP_MAX_PAYLOAD];
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            result = socket.recv_from(&amp;mut buf) =&gt; {
                let recv_us = sim_start.elapsed().as_micros() as u64;
                match result {
                    Err(e) =&gt; { tracing::error!(&quot;uplink_rx: recv error: {}&quot;, e); continue; }
                    Ok((len, src)) =&gt; {
                        // Deserialize
                        let cmd: CommandPacket = match bincode::deserialize(&amp;buf[..len]) {
                            Ok(c)  =&gt; c,
                            Err(e) =&gt; { tracing::warn!(&quot;uplink_rx: deser error: {}&quot;, e); continue; }
                        };
                        let dispatch_latency = recv_us - cmd.timestamp_us;
                        tracing::info!(cmd=?cmd.cmd_type, seq=cmd.seq_no,
                                      dispatch_latency_us=dispatch_latency, &quot;uplink_rx: received&quot;);
                        // Execute command based on type
                        handle_command(cmd, &amp;state).await;
                        heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
                    }
                }
            }
        }
    }
}
 
// Safety interlock: only execute commands that are valid for current state.
async fn handle_command(cmd: CommandPacket, state: &amp;Arc&lt;Mutex&lt;SystemState&gt;&gt;) {
    let current = { state.lock().await.clone() };
    match (cmd.cmd_type, current) {
        (CommandType::EmergencyStop, _)         =&gt; { /* always execute */ }
        (CommandType::SafeMode, _)              =&gt; { /* always execute */ }
        (CommandType::ResetSensor, SystemState::Fault) =&gt; { /* reset from fault */ }
        (_, SystemState::Fault) =&gt; {
            tracing::warn!(cmd=?cmd.cmd_type, &quot;INTERLOCK: command blocked — system in Fault state&quot;);
            return;
        }
        _ =&gt; { /* execute normally */ }
    }
    // Update state as appropriate
    tracing::info!(cmd=?cmd.cmd_type, &quot;command executed&quot;);
}


3.7 fault.rs — Fault Injection + Recovery Engine
ℹ Wk6+Wk10: Fault injection maps to Week 10 (error detection and recovery) and Week 6 (Circuit Breaker, Supervisor). The satellite must inject a fault every 60 seconds, detect it within the system, and recover within 200ms. If recovery exceeds 200ms, simulate mission abort.


pub enum CircuitState { Closed, Open(Instant), HalfOpen }
 
pub struct FaultEngine {
    pub circuit: CircuitState,
    pub consecutive_faults: u32,
    pub last_fault_at: Option&lt;Instant&gt;,
    pub total_faults: u64,
    pub total_recoveries: u64,
    pub max_recovery_ms: u64,
}
 
pub async fn run_fault_injector(
    buffer:    Arc&lt;Mutex&lt;SensorBuffer&gt;&gt;,
    state:     Arc&lt;Mutex&lt;SystemState&gt;&gt;,
    fault_tx:  tokio::sync::mpsc::Sender&lt;FaultPacket&gt;,  // send fault to downlink
    sim_start: Arc&lt;Instant&gt;,
    cancel:    CancellationToken,
    heartbeat: Arc&lt;AtomicU64&gt;,
) {
    let mut interval = tokio::time::interval(Duration::from_secs(FAULT_INJECT_INTERVAL_S));
    interval.tick().await; // skip first immediate tick
    let mut engine = FaultEngine { circuit: CircuitState::Closed, .. };
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            _ = interval.tick() =&gt; {}
        }
 
        let fault_start = Instant::now();
        let fault_type = pick_random_fault(); // alternate between DelayedSensor and CorruptedData
 
        tracing::warn!(fault=?fault_type, &quot;FAULT INJECTED&quot;);
 
        match fault_type {
            FaultType::DelayedSensor =&gt; {
                // Simulate a sensor delay by holding the buffer lock for 50ms
                tokio::time::sleep(Duration::from_millis(50)).await;
            }
            FaultType::CorruptedData =&gt; {
                // Corrupt the next packet in the buffer: flip bits in payload
                let mut buf = buffer.lock().await;
                // pop, corrupt, re-push
            }
            _ =&gt; {}
        }
 
        // Send fault notification to GCS via fault_tx channel
        let fault_pkt = FaultPacket {
            seq_no: engine.total_faults as u32,
            timestamp_us: sim_start.elapsed().as_micros() as u64,
            fault_type, affected_sensor: SensorId::Thermal,
            severity: 2, payload: [0u8; 32]
        };
        let _ = fault_tx.send(fault_pkt).await;
 
        // Recovery: wait for state to return to Nominal, up to FAULT_RECOVERY_LIMIT_MS
        let recovery_result = tokio::time::timeout(
            Duration::from_millis(FAULT_RECOVERY_LIMIT_MS),
            wait_for_nominal(&amp;state)
        ).await;
 
        let recovery_ms = fault_start.elapsed().as_millis() as u64;
        engine.max_recovery_ms = engine.max_recovery_ms.max(recovery_ms);
 
        match recovery_result {
            Ok(_) =&gt; {
                tracing::info!(recovery_ms, &quot;FAULT RECOVERED&quot;);
                engine.circuit = CircuitState::Closed;
            }
            Err(_) =&gt; {
                tracing::error!(recovery_ms, &quot;RECOVERY EXCEEDED 200ms — MISSION ABORT&quot;);
                *state.lock().await = SystemState::MissionAbort;
                engine.circuit = CircuitState::Open(Instant::now());
            }
        }
        engine.total_faults += 1;
        heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
    }
    tracing::info!(total_faults=engine.total_faults, max_recovery_ms=engine.max_recovery_ms,
                   &quot;fault_injector final stats&quot;);
}


3.8 watchdog.rs — Software Watchdog (Satellite)
⚠ RULE [Wk6 Watchdog]: Every task must call heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed) at the END of each loop iteration. The watchdog checks all heartbeats every WATCHDOG_CHECK_INTERVAL_S (1s). If any heartbeat is older than WATCHDOG_TIMEOUT_S (5s), log an alert. In a real system this would trigger a task restart — for simulation, log the stale task name and increment a stale_count.

pub async fn run_watchdog(
    heartbeats: Vec&lt;(&amp;&apos;static str, Arc&lt;AtomicU64&gt;)&gt;,
    sim_start:  Arc&lt;Instant&gt;,
    cancel:     CancellationToken,
) {
    let mut interval = tokio::time::interval(Duration::from_secs(WATCHDOG_CHECK_INTERVAL_S));
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            _ = interval.tick() =&gt; {
                let now_s = sim_start.elapsed().as_secs();
                for (name, hb) in &amp;heartbeats {
                    let last = hb.load(Ordering::Relaxed);
                    if now_s.saturating_sub(last) &gt; WATCHDOG_TIMEOUT_S {
                        tracing::error!(task=name, last_seen_s=last, now_s,
                                       &quot;WATCHDOG: task is STALE — would restart&quot;);
                    }
                }
            }
        }
    }
}

4.2 telemetry_rx.rs — Telemetry Reception + Decoding
This is the most metrics-dense module. It receives all UDP packets from the satellite, measures latency and drift, detects packet loss, and triggers the Dead Man&apos;s Switch.

🔴 CRITICAL: Use Instant::now() the INSTANT recv_from() returns — before ANY other processing. This is the most accurate receive timestamp. Never compute it after deserialization.


pub async fn run_telemetry_rx(
    bind_addr:    SocketAddr,    // 0.0.0.0:8080
    state:        Arc&lt;Mutex&lt;GcsSystemState&gt;&gt;,
    fault_tx:     tokio::sync::mpsc::Sender&lt;FaultPacket&gt;,
    cmd_queue:    Arc&lt;Mutex&lt;BinaryHeap&lt;PrioritizedCommand&gt;&gt;&gt;,
    sim_start:    Arc&lt;Instant&gt;,
    cancel:       CancellationToken,
    heartbeat:    Arc&lt;AtomicU64&gt;,
) {
    let socket = UdpSocket::bind(bind_addr).await.expect(&quot;GCS RX bind failed&quot;);
    let mut buf = [0u8; UDP_MAX_PAYLOAD];
 
    // Track per-sensor sequence numbers for gap detection
    let mut last_seq: HashMap&lt;SensorId, u32&gt; = HashMap::new();
    let mut consecutive_gap: u32 = 0;
    let mut loss_of_contact = false;
 
    // Per-sensor latency histograms
    let mut lat_hist: HashMap&lt;SensorId, Histogram&lt;u64&gt;&gt; = HashMap::new();
    for id in [SensorId::Thermal, SensorId::Power, SensorId::Imu] {
        lat_hist.insert(id, Histogram::&lt;u64&gt;::new(3).unwrap());
    }
 
    // Reception drift tracking
    let mut last_recv_us: HashMap&lt;SensorId, u64&gt; = HashMap::new();
 
    loop {
        // Dead Man&apos;s Switch: expect a packet within 200ms
        let recv_result = tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            r = tokio::time::timeout(
                Duration::from_millis(200),
                socket.recv_from(&amp;mut buf)
            ) =&gt; r
        };
 
        let recv_us = sim_start.elapsed().as_micros() as u64; // timestamp IMMEDIATELY
 
        match recv_result {
            Err(_timeout) =&gt; {
                // 200ms silence = danger (Week 8: Dead Man&apos;s Switch)
                consecutive_gap += 1;
                tracing::warn!(consecutive_gap, &quot;DMS: no packet for 200ms&quot;);
                if consecutive_gap &gt;= GCS_PACKET_LOSS_ALERT {
                    loss_of_contact = true;
                    tracing::error!(&quot;LOSS OF CONTACT — &gt;= 3 consecutive failures&quot;);
                    // Enqueue re-request command to uplink
                    enqueue_re_request(&amp;cmd_queue).await;
                }
                continue;
            }
            Ok(Err(e)) =&gt; { tracing::error!(&quot;recv_from error: {}&quot;, e); continue; }
            Ok(Ok((len, _src))) =&gt; {
                consecutive_gap = 0;
                loss_of_contact = false;
 
                // Enforce 3ms decode deadline (Week 8 + assignment spec)
                let decode_result = tokio::time::timeout(
                    Duration::from_millis(TELEMETRY_DECODE_MS),
                    async { bincode::deserialize::&lt;TelemetryPacket&gt;(&amp;buf[..len]) }
                ).await;
 
                let packet = match decode_result {
                    Ok(Ok(p))  =&gt; p,
                    Ok(Err(e)) =&gt; { tracing::warn!(&quot;decode error: {}&quot;, e); continue; }
                    Err(_)     =&gt; { tracing::warn!(&quot;DECODE EXCEEDED 3ms deadline&quot;); continue; }
                };
 
                // Route fault packets to fault manager
                if packet.packet_type == PacketType::FaultNotify {
                    // deserialize as FaultPacket and forward
                    continue;
                }
 
                // One-way latency = recv_us - packet.timestamp_us
                let latency_us = recv_us.saturating_sub(packet.timestamp_us);
                if let Some(hist) = lat_hist.get_mut(&amp;packet.sensor_id) {
                    hist.record(latency_us).ok();
                }
 
                // Reception drift = time since last packet of same type - expected period
                if let Some(last_us) = last_recv_us.get(&amp;packet.sensor_id) {
                    let expected_period_us = match packet.sensor_id {
                        SensorId::Thermal =&gt; THERMAL_PERIOD_MS * 1000,
                        SensorId::Power   =&gt; POWER_PERIOD_MS   * 1000,
                        SensorId::Imu     =&gt; IMU_PERIOD_MS     * 1000,
                    } as u64;
                    let actual_interval = recv_us - last_us;
                    let drift_us = actual_interval as i64 - expected_period_us as i64;
                    tracing::info!(
                        sensor=?packet.sensor_id, latency_us, drift_us,
                        seq=packet.seq_no, value=packet.value, &quot;telemetry_rx&quot;
                    );
                }
                last_recv_us.insert(packet.sensor_id, recv_us);
 
                // Sequence gap detection
                if let Some(&amp;prev_seq) = last_seq.get(&amp;packet.sensor_id) {
                    let gap = packet.seq_no.saturating_sub(prev_seq + 1);
                    if gap &gt; 0 {
                        tracing::warn!(sensor=?packet.sensor_id, expected=prev_seq+1,
                                      got=packet.seq_no, gap, &quot;PACKET LOSS DETECTED&quot;);
                        enqueue_re_request(&amp;cmd_queue).await;
                    }
                }
                last_seq.insert(packet.sensor_id, packet.seq_no);
 
                heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
            }
        }
    }
    // Print final histogram summary for all sensors
}


4.3 uplink_tx.rs — Command Uplink Scheduler
Manages a BinaryHeap of prioritised commands. Dispatches them to the satellite via UDP with a strict 2ms deadline for urgent/emergency commands.

PrioritizedCommand Struct

#[derive(Debug, Clone)]
pub struct PrioritizedCommand {
    pub packet:    CommandPacket,
    pub enqueue_us: u64,  // when command was enqueued — for queueing latency metric
}
 
// BinaryHeap ordering: lower priority number = pop first
impl Ord for PrioritizedCommand {
    fn cmp(&amp;self, other: &amp;Self) -&gt; Ordering {
        other.packet.priority.cmp(&amp;self.packet.priority)
    }
}
impl PartialOrd for PrioritizedCommand { fn partial_cmp(&amp;self, other: &amp;Self) -&gt; Option&lt;Ordering&gt; { Some(self.cmp(other)) } }
impl PartialEq  for PrioritizedCommand { fn eq(&amp;self, other: &amp;Self) -&gt; bool { self.packet.priority == other.packet.priority } }
impl Eq for PrioritizedCommand {}


run_uplink_tx() Logic

pub async fn run_uplink_tx(
    sat_addr:  SocketAddr,    // 192.168.1.10:9090
    cmd_queue: Arc&lt;Mutex&lt;BinaryHeap&lt;PrioritizedCommand&gt;&gt;&gt;,
    state:     Arc&lt;Mutex&lt;GcsSystemState&gt;&gt;,
    sim_start: Arc&lt;Instant&gt;,
    cancel:    CancellationToken,
    heartbeat: Arc&lt;AtomicU64&gt;,
) {
    let socket = UdpSocket::bind(&quot;0.0.0.0:0&quot;).await.unwrap();
    let mut interval = tokio::time::interval(Duration::from_millis(5)); // check queue every 5ms
    let mut tx_seq: u32 = 0;
    let mut deadline_misses: u64 = 0;
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            _ = interval.tick() =&gt; {}
        }
 
        let cmd = { cmd_queue.lock().await.pop() };
        let cmd = match cmd { Some(c) =&gt; c, None =&gt; continue };
 
        // Safety interlock: block non-emergency commands in Fault/MissionAbort state
        let gcs_state = { state.lock().await.clone() };
        if matches!(gcs_state, GcsSystemState::InterlockActive | GcsSystemState::LossOfContact) {
            if cmd.packet.priority &gt; 1 {
                tracing::warn!(cmd=?cmd.packet.cmd_type,
                               reason=&quot;interlock_active&quot;,
                               &quot;COMMAND REJECTED&quot;);
                continue;
            }
        }
 
        let dispatch_start = Instant::now();
        let deadline_ms = CMD_DISPATCH_MS; // 2ms
 
        let mut pkt = cmd.packet;
        pkt.seq_no = tx_seq;
        pkt.timestamp_us = sim_start.elapsed().as_micros() as u64;
 
        let bytes = bincode::serialize(&amp;pkt).unwrap();
 
        let send_result = tokio::time::timeout(
            Duration::from_millis(deadline_ms),
            socket.send_to(&amp;bytes, sat_addr)
        ).await;
 
        let dispatch_us = dispatch_start.elapsed().as_micros() as u64;
        let queue_latency_us = pkt.timestamp_us - cmd.enqueue_us;
 
        match send_result {
            Ok(Ok(_)) =&gt; {
                tracing::info!(cmd=?pkt.cmd_type, dispatch_us, queue_latency_us,
                               seq=tx_seq, &quot;uplink_tx: sent&quot;);
            }
            _ =&gt; {
                deadline_misses += 1;
                tracing::warn!(cmd=?pkt.cmd_type, dispatch_us,
                               &quot;uplink_tx: DISPATCH DEADLINE MISSED&quot;);
            }
        }
        tx_seq += 1;
        heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
    }
    tracing::info!(deadline_misses, &quot;uplink_tx final stats&quot;);
}


4.4 fault_mgr.rs — Fault Management + Interlock

pub async fn run_fault_mgr(
    fault_rx:  tokio::sync::mpsc::Receiver&lt;FaultPacket&gt;,
    state:     Arc&lt;Mutex&lt;GcsSystemState&gt;&gt;,
    cmd_queue: Arc&lt;Mutex&lt;BinaryHeap&lt;PrioritizedCommand&gt;&gt;&gt;,
    sim_start: Arc&lt;Instant&gt;,
    cancel:    CancellationToken,
    heartbeat: Arc&lt;AtomicU64&gt;,
) {
    let mut total_faults = 0u64;
    let mut interlock_latencies: Vec&lt;u64&gt; = Vec::new();
 
    loop {
        tokio::select! {
            _ = cancel.cancelled() =&gt; break,
            fault = fault_rx.recv() =&gt; {
                let fault = match fault { Some(f) =&gt; f, None =&gt; break };
                let detect_us = sim_start.elapsed().as_micros() as u64;
 
                tracing::warn!(fault=?fault.fault_type, severity=fault.severity,
                               &quot;fault_mgr: fault received&quot;);
 
                // Activate interlock: block non-emergency uplink
                {
                    let mut s = state.lock().await;
                    *s = GcsSystemState::InterlockActive;
                }
 
                let interlock_us = sim_start.elapsed().as_micros() as u64;
                let interlock_latency = interlock_us - detect_us;
                interlock_latencies.push(interlock_latency);
 
                tracing::info!(interlock_latency_us=interlock_latency,
                               &quot;fault_mgr: interlock APPLIED&quot;);
 
                // Critical alert if interlock took &gt; 100ms
                if interlock_latency &gt; GCS_INTERLOCK_LIMIT_MS * 1000 {
                    tracing::error!(interlock_latency_us=interlock_latency,
                                   &quot;CRITICAL GROUND ALERT: interlock exceeded 100ms&quot;);
                }
 
                // Enqueue SafeMode command
                let safe_cmd = PrioritizedCommand {
                    packet: CommandPacket { seq_no: 0, timestamp_us: detect_us,
                                            cmd_type: CommandType::SafeMode,
                                            priority: 1, payload: [0u8; 32] },
                    enqueue_us: detect_us,
                };
                cmd_queue.lock().await.push(safe_cmd);
 
                // Auto-resolve after 10s (simulated recovery window)
                tokio::time::sleep(Duration::from_secs(10)).await;
                { *state.lock().await = GcsSystemState::Nominal; }
                tracing::info!(&quot;fault_mgr: interlock CLEARED&quot;);
 
                total_faults += 1;
                heartbeat.store(sim_start.elapsed().as_secs(), Ordering::Relaxed);
            }
        }
    }
    let avg_interlock = if interlock_latencies.is_empty() { 0 }
        else { interlock_latencies.iter().sum::&lt;u64&gt;() / interlock_latencies.len() as u64 };
    tracing::info!(total_faults, avg_interlock_us=avg_interlock,
                   &quot;fault_mgr final stats&quot;);
}

6.2 Handling UDP Unreliability in Application Layer
Since UDP provides no guarantees, the application layer must compensate. Implement these three mechanisms:

Mechanism 1 — Sequence Numbers (seq_no field in every packet)
Every TelemetryPacket and CommandPacket has a seq_no: u32 that increments by 1 per packet, per sensor.
The GCS receiver tracks last_seq per SensorId. If recv seq_no &gt; last_seq + 1, packets were lost.
Number of lost packets = recv_seq - last_seq - 1. Log this gap with sensor ID and timestamps.
Do NOT wait or block on a gap. Log it, enqueue a re-request command, and continue processing.

Mechanism 2 — Timestamps (timestamp_us in every packet)
The sender records Instant as microseconds since sim_start and embeds it in the packet.
The receiver records its own Instant at the moment recv_from() returns.
One-way latency = receiver_us - packet.timestamp_us. This is valid ONLY because both machines use Instant (monotonic) and the timestamp_us is the sender&apos;s elapsed time since a common reference — not wall-clock time.
Both machines must start from approximately the same moment for this to be meaningful. Use a shared countdown or just note in the report that this is a relative latency metric.

Mechanism 3 — Dead Man&apos;s Switch (200ms silence timeout)
The GCS expects at least one packet every 200ms from the satellite (thermal runs every 100ms).
Implemented with tokio::time::timeout(Duration::from_millis(200), socket.recv_from(...)).
On timeout: increment consecutive_gap counter. If &gt;= 3: trigger Loss of Contact state.
This is the satellite-equivalent: if OCS hasn&apos;t received a heartbeat command from GCS for 30s, log a communication warning (not abort — commands are optional, telemetry is mandatory).

6.4 Fault Packet Routing Over the Downlink Channel
Fault packets are sent on the SAME downlink socket as telemetry (satellite sends to GCS port 8080). They are distinguished by the packet_type field. The GCS telemetry_rx function must inspect the packet_type and route accordingly:

// In telemetry_rx after successful deserialization:
match packet.packet_type {
    PacketType::SensorData =&gt; {
        // Normal telemetry processing path
    }
    PacketType::FaultNotify =&gt; {
        // Deserialize as FaultPacket (the payload bytes contain a FaultPacket)
        // Send to fault_mgr via fault_tx channel
        if let Ok(fault_pkt) = bincode::deserialize::&lt;FaultPacket&gt;(&amp;buf[..len]) {
            let _ = fault_tx.send(fault_pkt).await;
        }
    }
    PacketType::Heartbeat =&gt; {
        // OCS alive signal — update last_satellite_heartbeat timestamp
        tracing::debug!(&quot;satellite heartbeat received&quot;);
    }
    _ =&gt; { tracing::warn!(&quot;unexpected packet type from satellite&quot;); }
}
